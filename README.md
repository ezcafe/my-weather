# My Weather

## Introduction

My Weather shows the next 5 days weather of selected city

## Getting started

Open terminal and run the following commands

### Run app

```
git clone https://github.com/ezcafe/my-weather.git
cd my-weather
yarn install
yarn start
```

### Run test

`yarn test`
